## How to use this module

For each phrase or term:
1. Say it out loud 3 times.
2. Use it in a sentence about your own tech talk.
3. Practice using it naturally in a mock presentation.

## Lessons Learned

| Phrase | Listen | Usage |
|--------|--------|-------|
| in hindsight | [Nghe →](https://youglish.com/pronounce/in%20hindsight/english) | "looking back" — used when explaining what you'd do differently |
| the root cause was | [Nghe →](https://youglish.com/pronounce/the%20root%20cause%20was/english) | specific, direct — not "we had some issues" |
| what we changed as a result | [Nghe →](https://youglish.com/pronounce/what%20we%20changed%20as%20a%20result/english) | the concrete action taken after the learning |
| the lesson we took from this | [Nghe →](https://youglish.com/pronounce/the%20lesson%20we%20took%20from%20this/english) | extracting a transferable principle from an experience |

## Q&A Handling

| Phrase | Listen | Usage |
|--------|--------|-------|
| that's a great question | [Nghe →](https://youglish.com/pronounce/that%27s%20a%20great%20question/english) | buys 1–2 seconds to think |
| let me make sure I understand your question | [Nghe →](https://youglish.com/pronounce/let%20me%20make%20sure%20I%20understand/english) | asks for clarification gracefully |
| I don't have a definitive answer on that | [Nghe →](https://youglish.com/pronounce/I%20don%27t%20have%20a%20definitive%20answer/english) | honest "I don't know" — professional phrasing |

| happy to follow up on that | [Nghe →](https://youglish.com/pronounce/happy%20to%20follow%20up/english) | commit to a follow-up without blocking the talk |
