## Week 1 Goal

Deliver a 5-minute practice version of Tech Talk #1 — LLM Architecture — including: opening hook, system walkthrough, one live demo narration, and a closing with one ask for the team. Record it.

**How to verify:** Listen back. Does the opening make you want to keep listening? Is the technical walkthrough clear? Was there a 3-second silence or longer? Fix and re-record.

## Week 2 Goal

Deliver a 5-minute practice version of Tech Talk #2 — Production AI Lessons Learned — including: one specific failure story with root cause and resolution, one metric before/after comparison, and a team roadmap recommendation.

**How to verify:** Ask a colleague to listen to the recording. If they can name the failure, the fix, and the lesson — the story landed.

## KPI 4 milestone

- **Before Tech Talk #1 (July):** Complete lessons 1–7, practice full talk 3 times, collect slide feedback from one colleague.
- **Before Tech Talk #2 (November):** Complete lessons 8–14, practice full talk 3 times, prepare 5-question Q&A simulation.

## If you need more time

If the goals don't feel comfortable after 2 weeks, extend by 3–5 days before moving on.
**Fluency over schedule.**

## Moving to the Certification Prep track

Before starting the Certification Prep track, check:
- [ ] I can deliver a 5-minute tech talk opening without notes
- [ ] I can narrate a live demo for 3 minutes without falling silent
- [ ] I have handled 3 practice Q&A scenarios out loud
- [ ] Tech Talk #1 slides draft is started
