## Session goal

Chain four different communication acts in 15 minutes — test fluency across all modes.

## Shadowing passage

> Today you will do four activities in sequence. First: give your standup update for today. Speak for sixty seconds, no more, no less. Include what you finished yesterday, what you are doing today, and one specific blocker. Stop at sixty seconds even if you have more to say. Second: write a Slack message to your tech lead explaining that you found a security issue during code review — a user ID from one session was appearing in another session's response. Keep it under 60 words, include severity, scope, and your recommended next action. Third: record a code review comment on an imaginary pull request where a colleague has used a raw SQL query instead of the ORM abstraction the codebase uses everywhere else. Be specific, kind, and explain the reasoning — not just 'use the ORM.' Fourth: record a thirty-second status update to a non-technical stakeholder explaining that you will miss a deadline by one day because of the security issue you found, and tell them what you need from them. Play all four recordings back. Which mode is weakest?

## Key phrases

| Phrase | IPA | Note |
|--------|-----|------|
| security issue | /sɪˈkjʊərɪti ˈɪʃuː/ | never bury this — surface it immediately and clearly |
| severity | /sɪˈvɛrɪti/ | how serious is it? Always quantify: low / medium / high / critical |
| miss a deadline | /mɪs ə ˈdɛdlaɪn/ | communicate early, with reason and new timeline — never silently |

## 30-minute protocol

**Minutes 1–5** — Read the passage silently twice. Focus on meaning, not pronunciation.

**Minutes 6–15** — Shadow aloud with the text visible. 4 full passes. Match rhythm and stress.

**Minutes 16–22** — Shadow without the text. 3 passes. Accept mistakes — keep moving.

**Minutes 23–28** — Record yourself once. Play it back immediately. Note one phrase to improve.

**Minutes 29–30** — Write the phrase you struggled with. Say it 10 times slowly, then at full speed.

## Reflection

Which of the four communication modes felt most uncomfortable? That is your priority to practice.
