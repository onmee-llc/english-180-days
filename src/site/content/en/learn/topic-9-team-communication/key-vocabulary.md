## How to use this module

For each phrase or term:
1. Say it out loud 3 times.
2. Use it in a sentence about your own work.
3. Practice using it naturally in an answer to a practice question.

**Goal:** Zero hesitation when these words come up in an interview.

## Daily drill

Pick 3–5 phrases from this list each day. Use each one out loud in a complete sentence before moving on.

---

## Standup & Status

| Term / Phrase | Usage Example |
|---------------|---------------|
| blocker | "I'm blocked on the API integration — waiting for credentials from the DevOps team." |
| dependency | "This feature has a dependency on the auth service being deployed first." |
| blocked on / waiting for | "I'm blocked on the PR review — I've tagged David but haven't heard back." |
| on track | "The migration is on track for the Friday deadline." |
| at risk | "The timeline is at risk because the API rate limit is lower than expected." |

## Code Review

| Term / Phrase | Usage Example |
|---------------|---------------|
| pull request / PR | "I opened a PR for the retry logic — it's ready for review." |
| diff | "The diff is small — only forty lines changed across two files." |
| merge | "Can we merge this before end of day? It's blocking the next task." |
| LGTM (looks good to me) | "LGTM — just one minor nit on the variable naming." |
| nit | "Nit: this variable name could be more descriptive." |
| suggestion | "I have a suggestion for simplifying the error handling here." |
| approve / request changes | "I'll approve once the test coverage is added for the edge case." |

## Async Communication

| Term / Phrase | Usage Example |
|---------------|---------------|
| async / sync | "Let's handle this async — no need for a meeting." |
| follow up | "I'll follow up on this after I finish the current sprint." |
| circle back | "Let me circle back on this tomorrow once I have the benchmark results." |
| stakeholder | "I need to align with the product stakeholder before we commit to this approach." |
| alignment | "We need alignment on the API contract before both teams start building." |
| sign-off | "I'll need sign-off from the security team before we deploy." |

## Meeting & Collaboration

| Phrase | When to Use |
|--------|-------------|
| "I'd like to propose..." | Presenting a solution |
| "The problem we're solving is..." | Framing the discussion |
| "I've considered two options..." | Showing you evaluated alternatives |
| "I'm open to other ideas." | Inviting collaboration |
| "Let me summarize what we agreed on." | Closing a discussion |
| "I'll take the action item to..." | Committing to a next step |
