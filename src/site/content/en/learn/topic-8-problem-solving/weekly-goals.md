## Week 1 Goal

Practice thinking out loud on an unfamiliar problem for 5 minutes. Record and review.

**How to verify:** Record yourself. If you would be satisfied hearing this in a real interview, you've hit the goal.

## Week 2 Goal

Answer all 5 practice questions using the structured approach: think aloud, state hypothesis, test, conclude.

**How to verify:** Do a full dry run — start the recording, ask yourself the question, answer as if it's real.

## If you need more time

If the goals don't feel comfortable after 2 weeks, extend by 3–5 days before moving on.
**Fluency over schedule.**

## Moving to the next topic

Before starting the next topic, check:
- [ ] I can answer "Tell me about yourself" in 90 seconds (this is every day, all 180 days)
- [ ] I can answer the 5 practice questions for this topic without preparation
- [ ] My answers feel natural, not memorized
