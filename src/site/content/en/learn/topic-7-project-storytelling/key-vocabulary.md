## How to use this module

For each phrase or term:
1. Say it out loud 3 times.
2. Use it in a sentence about your own work.
3. Practice using it naturally in an answer to a practice question.

**Goal:** Zero hesitation when these words come up in an interview.

## Daily drill

Pick 3–5 phrases from this list each day. Use each one out loud in a complete sentence before moving on.

---

## Setting the Scene (Situation & Task)

| Phrase | When to Use |
|--------|-------------|
| "The challenge we faced was..." | Opening the Situation — what made this hard |
| "The context was..." | Setting background quickly |
| "At the time, we were dealing with..." | Establishing urgency or complexity |
| "My responsibility was to..." | Transitioning from Situation to Task |
| "I was brought in to..." | Emphasizing your specific role |
| "The scope included..." | Defining boundaries of your work |

## Describing Your Actions

| Phrase | When to Use |
|--------|-------------|
| "I decided to approach it by..." | Showing intentional decision-making |
| "The first thing I did was..." | Starting the Action sequence |
| "I evaluated several options and chose..." | Demonstrating analytical thinking |
| "The main trade-off I considered was..." | Showing engineering maturity |
| "I collaborated with... to..." | Showing teamwork without losing your role |
| "One key decision was..." | Highlighting a critical moment |

## Delivering Results

| Phrase | When to Use |
|--------|-------------|
| "The outcome was..." | Transitioning to measurable result |
| "As a result..." | Connecting your action to impact |
| "We saw a X% improvement in..." | Quantifying the impact |
| "The system now handles..." | Showing ongoing value |
| "One thing I learned from this was..." | Adding reflection — shows growth |
| "If I were to do it again, I would..." | Showing self-awareness and honesty |

## Recovery & Failure Phrases

| Phrase | When to Use |
|--------|-------------|
| "What didn't go as planned was..." | Honest failure acknowledgment |
| "The lesson I took from that was..." | Growth framing |
| "In hindsight, I would have..." | Mature reflection |
| "Despite the setback, we managed to..." | Showing resilience |
| "That experience changed how I approach..." | Demonstrating learning |

## Transition & Flow Phrases

| Phrase | When to Use |
|--------|-------------|
| "Let me walk you through a specific example." | Starting a STAR story |
| "To give you more context..." | Adding needed background |
| "The key takeaway was..." | Wrapping up concisely |
| "That's a great question — let me think of the best example." | Buying time to choose the right story |
