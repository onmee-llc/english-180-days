## Session goal

Articulate your philosophy on automation and when it creates real value.

## Shadowing passage

> My automation philosophy starts with a simple rule: automate the second time, not the first. The first time you do something manually, you're still learning the problem shape. The second time, you have enough information to know whether this task is worth automating and how to automate it well. The cost of premature automation is a system that automates the wrong thing, with the wrong interface, that costs more to maintain than doing the task manually would have. When I evaluate an automation candidate, I ask three questions. First: is this task repetitive enough to justify the build cost? Automation that runs once a week needs a higher time savings per run to pay off than automation that runs a hundred times a day. Second: is the input deterministic enough? Automation handles clear, well-defined inputs well and fails unpredictably on edge cases. If the task requires significant judgment, partial automation with a human checkpoint is often better than full automation that fails silently. Third: what is the cost of automation failure? High-consequence automation — anything that publishes content publicly, sends emails, or modifies financial data — requires more rigorous testing, staged rollout, and circuit breakers than automation with easily reversible outcomes. My biggest automation lesson: the most valuable automations are the ones that free up human attention for work only humans can do, not the ones that are technically impressive.

## 30-minute protocol

**Minutes 1–5** — Read the passage silently twice. Focus on meaning, not pronunciation.

**Minutes 6–15** — Shadow aloud with the text visible. 4 full passes. Match rhythm and stress.

**Minutes 16–22** — Shadow without the text. 3 passes. Accept mistakes — keep moving.

**Minutes 23–28** — Record yourself once. Play it back immediately. Note one phrase to improve.

**Minutes 29–30** — Write the phrase you struggled with. Say it 10 times slowly, then at full speed.

## Anti-Translation Drill — Think in English *(5 min)*

Pick **one key phrase** from today's table. Explain what it means using **only English**.

- No Vietnamese. No dictionary.
- If you don't know a word, describe it: *"It's when you…"* / *"It's similar to…"* / *"The opposite would be…"*
- Then use the phrase in a new sentence about your own experience.

> Goal: Train circumlocution — the skill of explaining anything without knowing the exact word. This is what fluent speakers actually do.
