**Starter vocabulary:**

| Term | Definition |
|------|-----------|
| benchmark | a standardized test used to measure a specific model capability |
| eval set | a curated collection of test cases used to measure model behavior on a specific task |
| ground truth | the known-correct expected output for a test case |
| false positive | the model flagged or refused something it should have allowed |
| false negative | the model allowed or produced something it should have refused |
| calibration | the alignment between a model's stated confidence and its actual accuracy |
| human eval | evaluation where humans rate model outputs rather than comparing to a fixed expected answer |
| adversarial example | an input crafted to cause the model to fail in a specific way |
