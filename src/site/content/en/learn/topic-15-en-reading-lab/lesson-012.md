**Starter vocabulary:**

| Term | Definition |
|------|-----------|
| engineering velocity | the speed at which a team ships features and resolves issues |
| code review | the process of examining code changes before they are merged |
| pair programming | two engineers working on the same code at the same time |
| context window | the maximum amount of text an LLM can process in one request |
| hallucination in code | when an LLM generates plausible-looking but incorrect or non-existent code |
| autonomous coding agent | an LLM that can execute code, run tests, and iterate toward a solution without human intervention at each step |
| technical debt | accumulated shortcuts in code that slow down future development |
