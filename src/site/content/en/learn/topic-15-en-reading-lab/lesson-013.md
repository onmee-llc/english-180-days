**Starter vocabulary:**

| Term | Definition |
|------|-----------|
| REST (Representational State Transfer) | an API design style using HTTP methods and resource URLs |
| GraphQL | a query language for APIs where the client specifies exactly what data it needs |
| gRPC | a high-performance RPC framework using Protocol Buffers — common in microservices |
| webhook | a server-to-server callback — the server sends data to a registered URL when an event occurs |
| idempotent | an operation that produces the same result if applied multiple times |
| pagination | returning a subset of results per request to avoid overwhelming the client |
| versioning | managing API changes (e.g., /v1/, /v2/) so existing clients are not broken |
| contract | the agreed interface between an API provider and its consumers |
