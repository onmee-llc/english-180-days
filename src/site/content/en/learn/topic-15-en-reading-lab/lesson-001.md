**Starter vocabulary for CDN articles:**

| Term | Definition |
|------|-----------|
| edge node | a server located close to the user in a CDN network |
| origin server | the primary server that holds the canonical copy of content |
| cache hit | when requested content is found in the edge cache |
| cache miss | when requested content is not cached and must be fetched from origin |
| TTL (Time To Live) | how long a cached object remains valid before requiring revalidation |
| geographic distribution | the spreading of servers across multiple regions or countries |
| static asset | content that does not change per request — images, CSS, JS files |
