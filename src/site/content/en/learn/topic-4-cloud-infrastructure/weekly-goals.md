## Week 1 Goal

Walk through the cloud architecture of a real project you built — out loud, 4 minutes, no notes.

**How to verify:** Record yourself. If you would be satisfied hearing this in a real interview, you've hit the goal.

## Week 2 Goal

Answer the 5 practice questions. Each answer: structured, specific, uses real project examples.

**How to verify:** Do a full dry run — start the recording, ask yourself the question, answer as if it's real.

## If you need more time

If the goals don't feel comfortable after 2 weeks, extend by 3–5 days before moving on.
**Fluency over schedule.**
